# Bakery-Sales-Dashboard
## Project Overview
This project analyses bakery sales performance across multiple European cities using Python data visualisation techniques and an interactive Streamlit dashboard.

## Technologies Used
- Python
- Pandas
- Matplotlib
- Seaborn
- Plotly
- Streamlit

## Features
- Data Cleaning & Preprocessing
- Exploratory Data Analysis (EDA)
- Interactive Dashboard
- KPI Metrics
- Profitability Analysis
- Revenue Trend Analysis
- Heatmaps & Scatter Plots

## Dashboard
Interactive Streamlit dashboard developed for business decision-making and performance monitoring.
